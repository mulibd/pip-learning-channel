# Episode Template — מבנה קבוע ל-60 שניות

כל פרק עוקב אחרי אותו מבנה 6 סצנות. רק התוכן (אות/מספר/צבע + מילה קשורה) משתנה.

## משתנים לכל פרק
- TOPIC_TYPE: letter / number / color
- TOPIC_VALUE: למשל "A" או "1" או "Red"
- LINKED_WORD: מילה שמדגימה (למשל "Apple" עבור A)

## מבנה הסצנות

**סצנה 1 (0-8s) — Intro/Hook**
Pip קופץ למסך עם ריקוד קצר. טקסט גדול "[TOPIC_TYPE] [TOPIC_VALUE]!" קופץ פנימה.
מוזיקה: ג'ינגל פתיחה קבוע (זהה בכל פרק).

**סצנה 2 (8-20s) — הצגת הנושא**
[TOPIC_VALUE] גדול, צבעוני, מסתובב/קופץ למסך. Pip מצביע ושר את הערך 3 פעמים עם תנועת יד.

**סצנה 3 (20-35s) — קישור למילה**
[LINKED_WORD] קופץ למסך. Pip מתעסק איתו (נושך/מחבק/מצביע, תלוי בנושא).
שיר: "[TOPIC_VALUE] is for [LINKED_WORD]!" עם חזרה x2.

**סצנה 4 (35-48s) — תרגול אינטראקטיבי**
3 אפשרויות מופיעות על המסך, Pip "מחפש" ומוצא את [TOPIC_VALUE], קופץ משמחה.

**סצנה 5 (48-56s) — חזרה/חיזוק**
חזרה מהירה x2, קצב מואץ קלות, Pip רוקד.

**סצנה 6 (56-60s) — Outro**
Pip מנפנף, "See you next time!" + preview של הפרק הבא.

## כללי ברזל לעקביות
1. תמיד אותו ג'ינגל פתיחה/סיום (קובץ אודיו קבוע, לא נוצר מחדש כל פעם)
2. תמיד אותו תיאור Pip מ-PIP-CHARACTER.md
3. תמיד 60 שניות, תמיד 6 סצנות, תמיד אותו תזמון
4. שינוי יחיד מותר: TOPIC_VALUE + LINKED_WORD
